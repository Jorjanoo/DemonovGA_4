## 3. Выводы по сравнению результатов фильтрации

### Bounding Box фильтр
**Преимущества:**
- Самый прямой и быстрый контроль над областью
- Эффективно уменьшает объём данных для обработки
- Простота настройки параметров (минимальные и максимальные координаты)

**Недостатки:**
- Может «обрезать» полезные объекты, расположенные на границе бокса
- Не учитывает форму объектов, только их положение в пространстве

### Фильтр по высоте
**Преимущества:**
- Эффективно разделяет «землю» и «надземные» объекты
- Полезен для удаления точек земли при анализе сооружений или растительности
- Интуитивно понятный параметр (порог высоты)

**Ограничения:**
- На наклонной или неровной местности простой порог может давать ошибки
- Требует предварительного выравнивания данных или учёта рельефа
- Может исключать низкие, но важные объекты (бордюры, фундаменты)

### Радиусный (distance) фильтр
**Преимущества:**
- Идеален для локальной сегментации вокруг точки интереса
- Сохраняет форму объектов внутри радиуса
- Позволяет фокусироваться на конкретных зонах

**Недостатки:**
- Чувствителен к выбору центра и радиуса
- Может исключать части объектов, выходящие за пределы радиуса
- Не эффективен для выделения протяжённых или сложно-форменных объектов

## 4. Рекомендации по применению фильтров

### Оптимальные стратегии:

| Сценарий применения | Рекомендуемый подход |
|---------------------|---------------------|
| **Первичное уменьшение объёма данных** | `Bounding Box → Последующая фильтрация` |
| **Выделение надземных объектов** | `Фильтр по высоте → Уточнение другими методами` |
| **Локальный анализ конкретной зоны** | `Радиусный фильтр` |
| **Комплексная очистка данных** | `Bounding Box → Фильтр по высоте → Радиусная сегментация` |

### Комбинированные подходы:
1. **Двухэтапная фильтрация**:
   - Сначала Bounding Box для грубого отбора области
   - Затем радиусный фильтр для точной локализации объекта

2. **Последовательное уточнение**:
Исходные данные
- Bounding Box (снижение объёма)
- Фильтр по высоте (отделение объектов)
- Радиусный фильтр (фокусировка на цели)


3. **Адаптивная стратегия**:
- Для плоской местности: фильтр по высоте
- Для локальных объектов: радиусный фильтр  
- Для больших площадей: Bounding Box с последующим разбиением на зоны

## Заключение

Каждый фильтр имеет свою нишу применения:
- **Bounding Box** — для быстрого начального отбора
- **Фильтр по высоте** — для вертикальной сегментации
- **Радиусный фильтр** — для точечного анализа

**Комбинация методов** обычно даёт наилучшие результаты, позволяя сначала сократить объём данных, а затем точно выделить целевые объекты с учётом их пространственных характеристик.

---
*Примечание: Эффективность фильтрации зависит от характеристик исходных данных и конкретных целей анализа. Всегда рекомендуется визуальная проверка результатов после каждого этапа фильтрации.*
